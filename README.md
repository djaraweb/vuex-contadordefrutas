# Mi Primera Aplicación usando Vuex

## Descripcion

```
En esta aplicacion, se realizan practicas uzando vuex para el control de estados en Vue.
Para estilos del Frontend se ha tomado como maquedator Boostrap.
```

```

```
npm run build
